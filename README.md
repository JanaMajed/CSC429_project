# CSC 429 Project 
##Wareefa Dates


###Web applications now support shopping, education, and other daily services. As their usage increases, securing them has become critical. Poor security could lead to vulnerabilities, such as SQL injection, cross-site scripting, and unauthorized access.


###For this project, we created a web application called “Wareefa Dates,” designed to give users a premium experience when purchasing dates. The site features a welcoming homepage, intuitive login and registration pages, and dedicated dashboards for both users and administrators. We built the platform using PHP and MySQL.
